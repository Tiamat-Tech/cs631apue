CS631 -- Advanced Programming in the UNIX Environment - NetBSD

All software development will be done on a NetBSD system. It is your responsibility to prepare yourself for use of such a system prior to the start of the class.

All grading will be done on a NetBSD >= 10.0 system. While you may choose to develop on your own personal host, you should make sure that your code compiles and runs flawlessly on this OS.

Here are some options for you to use NetBSD for this class:

  • install the latest release of NetBSD on your laptop
  • install the latest release of NetBSD in a virtual machine such as VMWare Fusion or VirtualBox -- see these instructions for details
  • install the latest release of NetBSD in UTM on Apple M1 hardware (video instructions)
  • create a NetBSD AWS EC2 instance
  • create a NetBSD Linode instance
  • get a free shell account on a provider that offers NetBSD shell access, e.g., sdf.org
Determining the right solution for you and getting everything set up is considered a pre-requisite for this class.
