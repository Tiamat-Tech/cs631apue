CS631 -- Advanced Programming in the UNIX Environment - Linux Lab

In this class, it is frequently useful to compare how different Unix(-like) systems differ in their behavior. For this, it's useful for you to have an account on the shared linux-lab.cs.stevens-tech.edu systems. It is your responsibility to get an account on these systems prior to the start of the class.

In order to get an account on linux-lab.cs.stevens-tech.edu, you need to:

  • go to https://sit.teamdynamix.com/TDClient/1865/Portal/Requests/ServiceDet?ID=8231
  • select 'Contact CS Dept' on the right
  • fill in the request

Access to linux-lab.cs.stevens.edu is via ssh(1), using a load balancing mechanism that will drop you into any one of a handful of systems. As noted on the course website, you are expected to be familiar with operating in this environment.
